# Next Steps for the Portfolio

1) Add your photo
- Place an image at `images/me.jpg` and update `photoLabel`/`photoCaption` in `content.json`.

2) Publish content
- Fill `portfolio.units` with assignments (title, description, reflection).
- Add `reflection.entries` with learn/went well/challenging.

3) Tweak wording
- Update hero rotating words and tagline accents in `content.json` if you want different adjectives.
- Adjust phone/email display formats in `content.json`.

4) Optional sections to add
- Skills/progress bars
- Now Learning strip on Home
- Timeline of milestones
- Resume/Download link
- Social links (GitHub/email)

5) Accessibility/UX
- Consider a high-contrast or dark toggle.
- Add alt text to your photo when you replace the placeholder.

6) Publishing checklist
- Push changes to GitHub.
- Enable GitHub Pages (Settings → Pages → Deploy from branch → main → /root).
